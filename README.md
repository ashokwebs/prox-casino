## Overview

**PROX CASINO** is a fully offline, terminal-based casino simulator. No servers, no accounts, no real money вҖ” just pure gameplay in your terminal. Built with [ratatui](https://github.com/ratatui-org/ratatui) for a rich, responsive TUI experience.

### Why?

- рҹҺ° **Play anywhere** вҖ” works offline, no internet needed
- рҹ’ҫ **Your data, your machine** вҖ” saves locally via atomic JSON writes
- вҡЎ **Instant startup** вҖ” sub-100ms launch, native Rust performance
- рҹҺЁ **Beautiful TUI** вҖ” hand-crafted ASCII cards, animated slot reels, themed UI

## Quick Start

### Prerequisites

- [Rust](https://rustup.rs/) 1.75 or later

### Run from source

```bash
git clone https://github.com/ashokwebs/prox-casino.git
cd prox-casino
cargo run --release
```

### Install globally

```bash
cargo build --release
cp target/release/prox-casino ~/.local/bin/casino
```

Then just type `casino` from anywhere.

### Debug mode

```bash
RUST_LOG=info cargo run
```

### Tech Stack

| Component | Library |
|-----------|---------|
| UI Framework | [ratatui](https://github.com/ratatui-org/ratatui) 0.27 |
| Terminal Backend | [crossterm](https://github.com/crossterm-rs/crossterm) 0.27 |
| Async Runtime | [tokio](https://tokio.rs/) 1.38 |
| Serialization | [serde](https://serde.rs/) + serde_json |
| Database | [rusqlite](https://github.com/rusqlite/rusqlite) 0.31 *(future)* |
| Error Handling | [anyhow](https://github.com/dtolnay/anyhow) + [thiserror](https://github.com/dtolnay/thiserror) |

### Design Principles

- **Offline-first** вҖ” all game logic runs locally; online mode is scaffolded for future use
- **Atomic saves** вҖ” writes to `.tmp` then renames, preventing corruption
- **Server-authoritative ready** вҖ” security architecture designed so online mode can be added without client trust

## Save Data

Your progress is saved automatically to:

```
~/.local/share/prox-casino/offline_save.json
```

Tracked stats include:
- Chips balance & lifetime earnings
- Games played, win/loss/push counts
- Streaks (win, loss, push)
- Bust count, double-down count
- Slots: spins, biggest win, jackpot hits, symbol appearances
